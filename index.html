How to use

          

            

              1. Enter Regex Pattern: Use standard JavaScript regex syntax. Example: \w+@\w+\.\w+ for email
            

            

              2. Add Flags: Global (g) = find all matches, Case-insensitive (i) = ignore case, Multiline (m) = ^ $ per line
            

            

              3. Test String: Paste your text to test. Matches will highlight in real-time.
            

            

              Common Patterns: \d = digits, \w = word chars, \s = space, . = any char, * = 0+ times, + = 1+ times, ? = 0-1 times, {n,m} = n to m times
